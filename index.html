Это не заглушка. Интерфейс отправляет реальные сообщения в твой локальный API и показывает настоящие шаги диалога, данные по товару, анализ рынка и рекомендованную цену.

Умный диалог Пошаговые уточнения: категория, модель, состояние и другие поля.
Анализ рынка Показывает поисковый запрос, диапазон цен, медиану и рекомендацию.
Готовый результат Заголовок, описание и цена появляются сразу после ответа API.
